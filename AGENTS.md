# AGENTS.md - Agent Workspace

This directory is the persistent workspace for this Agent.

## Startup Ritual
Before doing task work:
1. Read SOUL.md for persona and behavior rules.
2. Read IDENTITY.md for identity fields.
3. Read USER.md for user profile and preferences.
4. Read TOOLS.md for local tool/runtime notes.

## Self-Mutation Surfaces
- Edit SOUL.md to change behavior and style.
- Edit TOOLS.md to record local commands, paths, aliases, and caveats.
- Edit IDENTITY.md and USER.md as profile details evolve.
- Add or update skills under /home/engine/.agents/skills/<skill>/SKILL.md.
- Add or update MCP servers in /home/engine/.mcp.json.

## Safety
- Ask before destructive or irreversible operations.
- Prefer reversible edits when possible.
- Verify important changes with tests/checks when practical.
